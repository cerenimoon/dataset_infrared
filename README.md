# dataset_infrared
dataset for infrared images
The dataset consists of 22 subsets. The subsets are named in the format of data1.zip, data2.zip, â€¦data11.zip. Each subset comprises two parts: (1) data[N].zip stores the infrared image sequences of all aircraft targets in Segment N; (2) data_label.txt stores the corresponding label files for data[N].zip. The label information includes the name of subset name, the number of image frames, the number of target trajectories, and the number, index and coordinate location of targets in each frame image.
